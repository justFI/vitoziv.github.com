Grid Diary（[http://griddiaryapp.com/]）

在 Grid Diary 中我主要贡献在 UI 交互方面的开发，Grid Diary 是非常注重细节的应用。Grid Diary 的交互细节也是它的一个亮点。在这个项目中最大的挑战是 UI 的适配和自定义交互视图的开发，因为 Grid Diary 是为数不多的全部界面都适配了横竖屏的应用！这就会要求所有自定义的 UI 都需要考虑通用性，在任何情况下都可以使用。经常会遇到一些功能因为适配问题而提升了不少复杂度。要实现这些通用性高的 UI 需要对整个视图结构有较深的理解。
