这个项目主要由三个人开发，我主要贡献在 UI 方面的开发，实现各种自定义交互控件，如：自定义交互转场、照片选择和查看组件、欢迎界面、教程引导系统等。因为这款应用支持横屏以及 iPad，在做这款应用时所有的 UI 控件都会考虑到在各种尺寸下的适配，虽然工作上增加了难度，但这也让我对 UI 实现上有了更深入的理解，熟知响应链、各种 UI 类的生命周期以及图层渲染原理。

两个比较重设计的天气类应用，我实现了部分的交互动画和自定义转场，由于这两个应用贴图很多，所以在开发中性能优化上花了不少功夫。并且大量用到了 CoreAnimation 中的内容，也让我对 CoreAnimation 有比较深入的研究。另外也封装了各种天气 API 。
